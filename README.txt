## Descripción del proyecto

SnifferGUI es una aplicación desarrollada en C++ que permite la captura, análisis y visualización de paquetes de red en tiempo real. El proyecto utiliza la biblioteca Npcap para acceder al tráfico de red y fue desarrollado utilizando el framework Qt para la construcción de la interfaz gráfica.
